# FleekChat

Try it live:  https://fleek.chat
Demo video: https://youtu.be/1Z6tnafYcxU

This is a Fleek react starter kit

it provides a login / register flow for fleek apps.

To try it out, 

1. clone the repo
2. run `npm install`
3. run `npm start`

Sit back and enjoy!
